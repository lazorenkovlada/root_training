Створення нового репозиторію
1. На сторінці профілю на GitHub натискаємо кнопку "new repository"
2. Вводимо назву та при бажанні додаємо опис, і далі натискаємо "Create repository"
3. Переходимо в термінал і вводимо git clone <посилання на репозиторій яке ми копіюємо після створення>
4. cd <назва папки репозиторію>
5. Копіюємо cp файли звідкись, або створюємо nano і т.д. Або можна touch назва_папки/.gitkeep
6. git add .
7. git commit -m "короткий коментар який описує які зміни ми внесли цього разу"
8. git push
